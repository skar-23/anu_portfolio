# Donipati Anu Kumari — Portfolio

Personal portfolio website for Donipati Anu Kumari, Machine Learning Engineer.

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion

## Development

```sh
npm install
npm run dev       # http://localhost:8080
npm run build
npm run lint
npm test
```
