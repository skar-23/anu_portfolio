# AGENTS.md

This file provides guidance to WARP (warp.dev) when working with code in this repository.

## Commands

```sh
npm run dev        # Start dev server on http://localhost:8080
npm run build      # Production build
npm run lint       # ESLint (TypeScript + React hooks rules)
npm test           # Run unit tests once (Vitest)
npm run test:watch # Run unit tests in watch mode
```

To run a single test file:
```sh
npx vitest run src/test/example.test.ts
```

E2E tests use Playwright via `playwright-fixture.ts` (powered by `lovable-agent-playwright-config`). Import `test`/`expect` from `./playwright-fixture` rather than directly from `@playwright/test`.

## Architecture

This is a single-page React portfolio site (no backend). React Router is configured in `src/App.tsx` with two routes: `/` → `pages/Index.tsx` and a catch-all `*` → `pages/NotFound.tsx`.

`pages/Index.tsx` is the entire site — it imports and stacks every section component in vertical order: `Navbar → HeroSection → EducationSection → SkillsSection → FreelancingSection → ProjectsSection → CodingPlatformsSection → CertificatesSection → AchievementsSection → ResumeSection → ContactSection → Footer`.

**All portfolio content is hardcoded inline** inside each section component (no CMS, no external API, no data files). To update text, links, or project details, edit the relevant component directly.

### Key shared primitives

- `SectionWrapper` — wraps every section with a Framer Motion `whileInView` scroll-reveal animation and `section-padding` spacing. Use this for any new section.
- `SectionTitle` — consistent heading component used at the top of each section.
- `src/lib/utils.ts` — exports only `cn()`, the standard `clsx` + `tailwind-merge` helper.

### Styling system

- All theme colors are CSS custom properties in `src/index.css` (HSL format, e.g. `hsl(var(--primary))`). Do not hardcode color values.
- Custom utility classes defined in `src/index.css`: `.glass-card` (frosted glassmorphism), `.frost-badge`, `.gradient-text`, `.gradient-accent`, `.gradient-primary`, `.glow-primary`, `.card-hover`, `.section-padding`. Prefer these over one-off inline styles.
- Fonts: **Space Grotesk** for headings (`font-heading`), **Inter** for body (`font-body`) — loaded from Google Fonts.
- The design is light-only. `darkMode: ["class"]` is present in `tailwind.config.ts` but a dark theme has not been implemented.

### shadcn/ui

Components in `src/components/ui/` are generated by shadcn/ui and should not be manually edited. To add a new shadcn component: `npx shadcn@latest add <component>`. The alias `@/components/ui` is configured in `components.json`.

### Lovable integration

The `lovable-tagger` Vite plugin (active in dev mode only) tags components for the Lovable visual editor. Do not remove it from `vite.config.ts`. The `@` path alias resolves to `src/`.
